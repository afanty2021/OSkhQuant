# 看海量化交易系统 - 开发版 (V2.1.5-dev)

## 开发版构建说明

### 构建信息
- **版本号**: 2.1.5-dev
- **构建日期**: 2025-12-13
- **更新通道**: dev (开发版)
- **基于稳定版**: V2.1.4

### 开发版特性

1. **版本标识**
   - 版本号包含 `-dev` 后缀
   - 更新通道设置为 `dev`
   - 构建日期更新为当前日期

2. **启动脚本**
   - `run_dev.bat` - Windows批处理启动脚本
   - `run_dev.py` - Python启动脚本
   - 自动环境检查和依赖验证
   - 开发环境变量自动设置

3. **环境变量**
   - `KHQUANT_DEV=1` - 标识开发环境
   - `KHQUANT_VERSION=2.1.5-dev` - 当前版本
   - `KHQUANT_CHANNEL=dev` - 更新通道

### 如何使用

#### 方法1: 使用批处理脚本 (推荐)
```bash
# Windows双击运行或在命令行执行
run_dev.bat
```

#### 方法2: 使用Python脚本
```bash
python run_dev.py
```

#### 方法3: 直接运行主程序
```bash
python GUIkhQuant.py
```

### 环境要求

1. **Python环境**
   - Python 3.7或更高版本
   - 建议使用虚拟环境

2. **依赖包**
   - PyQt5==5.15.11
   - pandas==2.3.1
   - numpy==2.3.2
   - matplotlib==3.10.0
   - 其他依赖见 requirements.txt

3. **MiniQMT (可选)**
   - 用于获取实时行情数据
   - xtquant 包需要从MiniQMT安装目录获取

1. **测试建议**

### 常见问题

1. **依赖包缺失**
   ```
   解决方案: pip install -r requirements.txt
   ```

2. **xtquant未找到**
   ```
   解决方案:
   - 确保已安装MiniQMT客户端
   - 将xtquant包路径添加到Python路径
   ```

3. **PyQt5启动失败**
   ```
   解决方案:
   - 检查是否安装了完整版PyQt5
   - 尝试重新安装: pip install PyQt5==5.15.11
   ```

### 验证构建

运行验证脚本检查开发版是否构建成功：
